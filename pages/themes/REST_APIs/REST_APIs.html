Overview of REST APIs

REST (Representational State Transfer) is an architectural style used for designing networked applications.
RESTful APIs are widely used in web and mobile applications for their simplicity and scalability.
REST APIs use standard HTTP methods for CRUD (Create, Read, Update, Delete) operations.
This makes it easier to integrate with various clients and platforms.

CRUD Operations and Their REST Equivalent

CRUD Operations and Their REST Equivalent

CRUD Operations and Their REST Equivalent

Create - POST
Use POST method to create a new resource.
Read - GET
Use GET method to retrieve a resource.
Update - PUT
Use PUT method to update an existing resource.
Delete - DELETE
Use DELETE method to remove a resource.

Making GET, POST, PUT, DELETE Requests Using JavaScript's Fetch API

Making GET, POST, PUT, DELETE Requests Using JavaScript's Fetch API

GET Request Example


            fetch('https://api.example.com/resource')
                .then(response => response.json())
                .then(data => console.log(data))
                .catch(error => console.error('Error:', error));

POST Request Example


            fetch('https://api.example.com/resource', {
                method: 'POST',
                headers: {
                    'Content-Type': 'application/json'
                },
                body: JSON.stringify({ name: 'New Resource' })
            })
            .then(response => response.json())
            .then(data => console.log(data))
            .catch(error => console.error('Error:', error));

PUT Request Example


            fetch('https://api.example.com/resource/1', {
                method: 'PUT',
                headers: {
                    'Content-Type': 'application/json'
                },
                body: JSON.stringify({ name: 'Updated Resource' })
            })
            .then(response => response.json())
            .then(data => console.log(data))
            .catch(error => console.error('Error:', error));

DELETE Request Example


            fetch('https://api.example.com/resource/1', {
                method: 'DELETE'
            })
            .then(response => console.log('Deleted'))
            .catch(error => console.error('Error:', error));
